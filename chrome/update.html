Tahir is updated! - Version 1.0.5

As salamu alaykum wa rahmatullah,

We pray you are benefiting from Tahir! This is an automated message letting you know Tahir has updated to the latest version on your computer.
  • Alt+K now unblurs & reblurs an image. After unbluring an image, press Alt+K again to reblur it.
  • Add websites to your whitelist/exception list to prevent Tahir from blurring those websites. To do this, go to the website, click the Tahir icon, and press "add to whitelist."
  • Tahir is available for download on Firefox.
  • Several technical improvements.

  • Important: Based on the way Tahir was previously coded, it theoretically had access to a user's browsing history in order to easily implement the whitelist feature (as developers we never saw a users browsing history, this was only for technical purposes to make the whitelist feature work). We have since modified the code and no longer require that access.
